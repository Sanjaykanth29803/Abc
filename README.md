Inventory KPI & ABC Analysis Dashboard

This analytical dashboard built using Microsoft Power BI provides a comprehensive view of inventory performance through ABC classification, KPI monitoring, supplier analysis, and inventory cost evaluation.

The dashboard highlights important inventory metrics such as Inventory Value, Total Items, Item Value, and Holding Cost, enabling organizations to evaluate inventory efficiency and cost management.

Key insights presented in the dashboard include:

ABC Inventory Classification to categorize items based on their value and importance, helping businesses focus on high-priority inventory items.
Inventory Status Distribution using Red, Green, and Amber indicators to quickly identify operational performance levels.
Monthly Actual Value Trends to monitor inventory value fluctuations over time.
Scrap Value Analysis to track financial losses caused by damaged or unused inventory.
KPI Performance Comparison showing target vs actual performance for key metrics such as stock audit accuracy, supplier delivery performance, reorder compliance, and inventory turnover.
Supplier Contribution by Department to analyze how suppliers support departments such as Finance, Planning, Production, and Stores.

The dashboard also includes plant-level KPI performance metrics such as:

Inventory Carrying Cost %
Reorder Point Compliance %
Scrap Cost %
Stock Audit Accuracy %
Supplier On-Time Delivery %

Interactive filters including ABC Class, Department, and Quarterly Time Periods allow users to perform detailed analysis and explore inventory performance across different operational dimensions.

This dashboard helps businesses to:

Optimize inventory classification and control
Monitor supplier performance
Track inventory holding and scrap costs
Evaluate operational KPIs
Improve overall inventory management strategy

Overall, this project demonstrates how Power BI can be used to build advanced inventory KPI dashboards for data-driven supply chain and inventory optimization.
